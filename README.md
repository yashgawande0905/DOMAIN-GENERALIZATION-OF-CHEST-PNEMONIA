# DOMAIN GENERALIZATION OF CHEST PNEUMONIA

## Overview
This project focuses on domain generalization techniques for chest pneumonia detection using machine learning models. It aims to improve model robustness across different data distributions and domains in medical imaging and signal processing.

## Features
- Signal classification for pneumonia detection
- Domain generalization algorithms
- Feature extraction and analysis
- Confusion matrix visualization

## Files
- `main.py`: Main script for running the project
- `signal_classification.py`: Signal processing and classification module
- `annexure_A_signals.png`: Signal visualization
- `annexure_B_features.csv`: Extracted features dataset
- `annexure_C_confusion_matrix.png`: Model evaluation results
- `archive_dataset/`: Dataset files (healthy and diseased signals)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yashgawande0905/DOMAIN-GENERALIZATION-OF-CHEST-PNEMONIA.git
   cd DOMAIN-GENERALIZATION-OF-CHEST-PNEMONIA
   ```

2. Install dependencies (if any):
   ```bash
   pip install -r requirements.txt
   ```
   *Note: Create a `requirements.txt` file with necessary Python packages.*

## Usage
Run the main script:
```bash
python main.py
```

For signal classification:
```bash
python signal_classification.py
```

## Dataset
The project uses chest X-ray or signal data for pneumonia detection. The `archive_dataset` folder contains:
- Diseased signals (diseased_0.txt to diseased_99.txt)
- Healthy signals (healthy_0.txt onwards)
- Metadata files (Data_Entry_2017.csv, BBox_List_2017.csv)

*Note: Large dataset files may need to be downloaded separately if not included.*

## Results
- Feature analysis: `annexure_B_features.csv`
- Confusion matrix: `annexure_C_confusion_matrix.png`
